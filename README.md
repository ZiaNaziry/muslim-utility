# Muslim Utilities

Your all-in-one Islamic companion — free tools for every Muslim.

## Tools

- Zakat Calculator
- Prayer Times
- Dhikr Counter
- Hijri Converter
- Qibla Finder

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy

Push to GitHub and import on [Vercel](https://vercel.com).
